# QuizMex

A quiz app where you play answering some questions about Mexico's history.

## Run Locally

Clone the project

```bash
  git clone https://github.com/alxmcr/fe-quizmex-react-ts
```

Go to the project directory

```bash
  cd fe-quizmex-react-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
